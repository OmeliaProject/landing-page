OMELIA

Nous vous proposons une application dotée d'une intelligence artificielle permettant de booster votre façon de présenter.
Découvrez Omelia, votre assistante qui sera toujours là pour vous, afin que vous puissiez donner le meilleur de vous-même lors de vos présentations.

Pour qui ?

Que vous soyez étudiant, employé, PDG, ou en recherche d'emploie, Omelia est une application faite pour vous.
Nous avons tous besoins un jour de passer en publique, pour une présentation ou un entretiens d'embauche par exemple.
Nous avons pensé notre application pour qu'elle s'adapte a vos besoins

L'Analyse

L’enregistrement est minutieusement analysé sur plusieurs aspects :

  • Le temps de la présentation
  • Le langage corporel
  • Les mots choisis
  • La tonalité utilisée

Le Conseil

Omelia est avant tout une application de conseil.
Une fois l'analyse terminée, une liste de vidéo vous permettant de progresser vous sera proposées.
Ces conseils sont personnalisés et en accord avec la présentation sur laquelle vous travaillez.

Le Prix

Omelia souhaite rester abordable.
Pour nous, tout le monde à le droit de s'entraîner.
Nous proposons don deux tarifs :

  • Un tarif gratuit ou vous pourrez profiter de 3 analyses par semaines
  • Et un tarif premium à seulement 3€ par mois vous permettant d'avoir des entraînements illimités
    ainsi qu'un retour plus approfondis

Pas de panique si vous souhaitez plus de 3 analyses. Pour chaque entraînement supplémentaire une courte vidéo vous sera proposer afin de recharger vos entraînements hebdomadaires.
